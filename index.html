Analisis Digital Forensik Serangan Web

Proyek Analisis Digital Forensik Serangan Web bertujuan untuk menyelidiki dan menganalisis serangan terhadap aplikasi web dengan menggunakan teknik digital forensik untuk mengidentifikasi dan memitigasi potensi ancaman. Dalam proyek ini, dilakukan pengumpulan bukti digital dari server, log sistem, dan jaringan untuk mendeteksi jejak serangan seperti SQL Injection, Cross-Site Scripting (XSS), dan Cross-Site Request Forgery (CSRF). Proses analisis mencakup pemetaan celah keamanan, identifikasi metode eksploitasi, serta penyusunan laporan forensik yang rinci. Proyek ini bertujuan untuk memberikan pemahaman mendalam tentang ancaman yang dihadapi aplikasi web dan menyarankan langkah-langkah perbaikan untuk meningkatkan keamanan aplikasi, serta melatih keterampilan dalam menggunakan alat forensik digital untuk investigasi keamanan.

Projek Pengukuran Suhu

Program ini mensimulasikan pengukuran suhu rata-rata dari tiga sensor di Bandung menggunakan model sistem terdistribusi dan paralel berbasis Publish-Subscribe. Sensor (publisher) mengirimkan data suhu secara acak setiap 10 detik ke broker MQTT, yang kemudian diteruskan ke subscriber. Subscriber menghitung suhu rata-rata berdasarkan jumlah data yang diterima. Program ini dibuat dengan Python dan library paho-mqtt, serta dapat dijalankan secara paralel di mesin atau proses berbeda, merepresentasikan sistem yang scalable dan efisien dalam pengolahan data sensor.

Projek Web Server Berbasis TCP Socket

Komunikasi antar perangkat dilakukan melalui protokol TCP/IP, memungkinkan sharing data antar komputer dalam jaringan.
